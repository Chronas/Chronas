Was ist Chronas?

Chronas ist eine Sprache die in der Java Laufzeitumgebung läuft, und einen vereinfachten Syntax besitzt.

Hört sich ja gut an aber was bringt mir die Programmiersprache?

  • Einfach für Programmieranfänger
  • spart Zeit
  • spart Code
